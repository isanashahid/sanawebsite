# OG Image Instructions

## Required Image
You need to add an Open Graph image for social media sharing.

### File Details
- **Filename**: `og-image.jpg`
- **Location**: Place in `/public/` directory
- **Dimensions**: 1200 x 630 pixels
- **Format**: JPG or PNG
- **File Size**: < 1MB recommended

### Design Recommendations
To match the neo-brutalism theme:
- Use dark background (#0a0a0a)
- Include your name in bold typography (Space Grotesk)
- Add neon accents (electric blue #00d4ff, hot pink #ff006e, lime green #adff00)
- Include your title: "AI GTM Strategist & Product Marketing Manager"
- Use thick borders (8px) around text boxes
- Keep it asymmetric and bold

### Tools to Create
- Canva (templates available)
- Figma (design from scratch)
- Adobe Photoshop
- Online OG image generators

### Optional: Additional Social Images
You can also add:
- `icon-192.png` (192x192) - PWA icon
- `icon-512.png` (512x512) - PWA icon
- `favicon.ico` - Browser tab icon (already exists)

Once you add `og-image.jpg`, it will automatically be used for:
- LinkedIn previews
- Twitter cards
- Facebook shares
- Other social platforms
